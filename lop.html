Laços de Repetição em C

Aprenda a repetir comandos de forma controlada.

Início Estrutura Variáveis Entrada/Saída Condicionais Laços

Quando usar laços?

Laços de repetição servem para executar o mesmo bloco de código várias vezes. Eles evitam repetição manual e deixam o programa mais flexível.

printf("1\n");
printf("2\n");
printf("3\n");

Com um laço, o mesmo resultado pode ser produzido de forma mais inteligente:

for (int i = 1; i <= 3; i++) {
    printf("%d\n", i);
}

1. for

O for é ideal quando você sabe quantas vezes o laço deve rodar.

for (int i = 0; i < 5; i++) {
    printf("i = %d\n", i);
}

Ele possui três partes: inicialização, condição e atualização.

for (inicializacao; condicao; atualizacao) {
    // comandos repetidos
}

2. while

Use o while quando a condição precisa ser verificada antes de cada repetição.

int contador = 0;

while (contador < 5) {
    printf("contador = %d\n", contador);
    contador++;
}

Se a condição começar falsa, o corpo do laço não será executado nenhuma vez.

3. do...while

O do...while garante pelo menos uma execução antes da verificação da condição.

int numero = 0;

do {
    printf("Digite 1 para sair: ");
    scanf("%d", &numero);
} while (numero != 1);

Esse laço repete até que o usuário digite 1.

Comparando os laços

Laço Melhor uso
for Quando a quantidade de repetições é conhecida.
while Quando depende de uma condição que pode mudar.
do...while Quando o bloco precisa rodar pelo menos uma vez.

4. Laços aninhados

Você pode colocar um laço dentro de outro. Isso é útil para matrizes, tabelas e desenhos no terminal.

for (int linha = 1; linha <= 3; linha++) {
    for (int coluna = 1; coluna <= 4; coluna++) {
        printf("(%d,%d) ", linha, coluna);
    }
    printf("\n");
}

Resultado:

(1,1) (1,2) (1,3) (1,4)
(2,1) (2,2) (2,3) (2,4)
(3,1) (3,2) (3,3) (3,4)

5. break e continue

Use break para sair imediatamente do laço e continue para pular a iteração atual.

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;
    }

    if (i % 2 == 0) {
        continue;
    }

    printf("i impar = %d\n", i);
}

Esse exemplo imprime apenas os números ímpares até que i chegue a 5.

6. Exemplo prático: soma de 1 a 10

int soma = 0;

for (int i = 1; i <= 10; i++) {
    soma += i;
}

printf("Soma = %d\n", soma);

O laço acumula cada valor de 1 a 10 dentro da variável soma.

7. Exemplo prático: tabuada

int numero = 7;

for (int i = 1; i <= 10; i++) {
    printf("%d x %d = %d\n", numero, i, numero * i);
}

Exercícios rápidos

  1. Mostre os números de 1 a 100 usando for.
  2. Leia números até o usuário digitar 0 e mostre a soma total.
  3. Faça uma tabuada usando o número digitado pelo usuário.
  4. Use laços aninhados para imprimir um retângulo de asteriscos.
Dica: verifique se a variável de controle muda dentro do laço. Caso contrário, o programa pode entrar em loop infinito.
