Bem-vindo ao tutorial de C

Este material foi organizado em tópicos dedicados para facilitar seu aprendizado. A linguagem C é uma ótima porta de entrada para entender lógica, variáveis, memória, compilação e construção de programas mais próximos do funcionamento real do computador.

  • Estrutura - Entenda como os programas em C são organizados.
  • Variáveis - Conheça os tipos de dados e como armazenar informações.
  • Entrada e Saída - Aprenda a interagir com o usuário.
  • Condicionais - Controle o fluxo de seus programas.
  • Laços - Domine as estruturas de repetição.

Como estudar este conteúdo

Uma boa forma de aprender C é seguir uma sequência simples: leia o conceito, copie o exemplo, compile, execute e depois altere pequenos detalhes para observar o resultado.

  1. Comece pela estrutura básica de um programa.
  2. Aprenda a declarar variáveis e escolher tipos de dados.
  3. Use printf e scanf para entrada e saída.
  4. Crie decisões com if, else e switch.
  5. Automatize repetições com for, while e do...while.

Primeiro programa completo

Este exemplo reúne os primeiros conceitos: biblioteca, função principal, variável, entrada, saída e retorno do programa.

#include <stdio.h>

int main(void) {
    int idade;

    printf("Digite sua idade: ");
    scanf("%d", &idade);

    printf("Voce tem %d anos.\n", idade);

    return 0;
}
Dica: pratique escrevendo e compilando pequenos programas. A linguagem C ajuda a entender como o computador organiza memória e processos.
