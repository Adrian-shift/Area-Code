Programação em C - Introdução

Uma página simples para aprender os conceitos básicos da linguagem C.

Início Laços

1. Estrutura de um programa em C

Um programa C básico usa a função main() como ponto de entrada. O código abaixo exibe uma mensagem no console:

#include <stdio.h>

int main(void) {
    printf("Olá, C!\n");
    return 0;
}

Explicação:

  • #include <stdio.h> traz a biblioteca padrão de entrada e saída.
  • int main(void) declara a função principal.
  • printf imprime texto na tela.
  • return 0; indica que o programa terminou com sucesso.

2. Variáveis e tipos básicos

As variáveis armazenam valores. Os tipos mais comuns são:

  • int - números inteiros
  • float - números de ponto flutuante
  • double - números de ponto flutuante mais precisos
  • char - um único caractere
int idade = 20;
float altura = 1.75f;
double peso = 68.5;
char letra = 'A';

3. Entrada e saída

Use printf para mostrar dados e scanf para ler dados do usuário:

#include <stdio.h>

int main(void) {
    int numero;
    printf("Digite um número: ");
    scanf("%d", &numero);
    printf("Você digitou: %d\n", numero);
    return 0;
}

Observação: scanf usa & antes da variável para passar o endereço.

4. Condicionais

O if permite tomar decisões:

int a = 10;
if (a > 0) {
    printf("a é positivo\n");
} else if (a < 0) {
    printf("a é negativo\n");
} else {
    printf("a é zero\n");
}

5. Laços de repetição

Use for, while e do...while para repetir ações:

for (int i = 0; i < 5; i++) {
    printf("i = %d\n", i);
}

int j = 0;
while (j < 5) {
    printf("j = %d\n", j);
    j++;
}
