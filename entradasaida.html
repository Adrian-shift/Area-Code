Entrada e saída com printf e scanf

Use printf para mostrar dados e scanf para ler dados do usuário:

#include <stdio.h>

int main(void) {
    int numero;
    printf("Digite um número: ");
    scanf("%d", &numero);
    printf("Você digitou: %d\n", numero);
    return 0;
}

Observação: scanf usa & antes da variável para passar o endereço.

Dica: Sempre use formatadores corretos em printf e scanf: %d para inteiros, %f para floats, %s para strings.
