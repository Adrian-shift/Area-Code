Entrada e Saída em C

Aprenda como interagir com o usuário através da entrada e saída de dados.

Início Estrutura Variáveis Entrada/Saída Condicionais Laços

Saída com printf

Use printf para mostrar mensagens e valores no terminal. Os formatadores indicam onde os valores aparecem.

int idade = 18;
float media = 7.5f;

printf("Idade: %d\n", idade);
printf("Media: %.1f\n", media);

Entrada com scanf

Use scanf para ler dados digitados pelo usuário. Na maioria dos tipos, coloque & antes da variável para passar seu endereço de memória.

#include <stdio.h>

int main(void) {
    int numero;

    printf("Digite um numero: ");
    scanf("%d", &numero);

    printf("Voce digitou: %d\n", numero);
    return 0;
}

Formatadores comuns

Formatador Tipo Exemplo
%d int scanf("%d", &idade);
%f float scanf("%f", &altura);
%lf double no scanf scanf("%lf", &peso);
%c char scanf(" %c", &opcao);
%s Texto sem espaços scanf("%s", nome);

Lendo mais de um valor

Você pode ler vários valores no mesmo scanf, desde que use os formatadores corretos e passe as variáveis na mesma ordem.

int horas;
float valor_hora;

printf("Digite horas trabalhadas e valor por hora: ");
scanf("%d %f", &horas, &valor_hora);

printf("Total: R$ %.2f\n", horas * valor_hora);

Cuidado ao ler caracteres

Ao ler um char, coloque um espaço antes de %c. Esse espaço faz o scanf ignorar quebras de linha deixadas por leituras anteriores.

char resposta;

printf("Deseja continuar? ");
scanf(" %c", &resposta);

printf("Resposta: %c\n", resposta);

Exercícios rápidos

  1. Leia dois números inteiros e mostre a soma.
  2. Leia o nome e a idade de uma pessoa e mostre uma frase com esses dados.
  3. Leia o preço de um produto e mostre o valor com desconto de 10%.
Dica: use formatadores corretos em printf e scanf: %d para inteiros, %f para floats e %s para textos sem espaços.
