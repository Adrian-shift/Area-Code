Variáveis e Tipos Básicos em C

Conheça como armazenar e usar dados em seus programas.

O que é uma variável?

Uma variável é um espaço na memória usado para guardar um valor. Em C, você precisa informar o tipo da variável antes de usá-la.

int idade = 20;
float altura = 1.75f;
double peso = 68.5;
char letra = 'A';

Tipos básicos de dados

Os tipos definem que tipo de valor pode ser armazenado e quanto espaço será usado na memória.

Tipo Uso comum Exemplo
int Números inteiros int pontos = 10;
float Números decimais simples float media = 8.5f;
double Números decimais mais precisos double salario = 2500.75;
char Um caractere char opcao = 'S';

Declaração, atribuição e atualização

Declarar é criar a variável. Atribuir é colocar um valor nela. Atualizar é trocar o valor durante o programa.

int contador;      // declaração
contador = 1;      // atribuição
contador = 2;      // atualização

printf("Contador: %d\n", contador);

Constantes

Use const quando um valor não deve mudar depois de definido. Isso torna o código mais claro e evita alterações acidentais.

const double PI = 3.14159;
double raio = 2.0;
double area = PI * raio * raio;

printf("Area: %.2f\n", area);

Nomes de variáveis

Escolha nomes claros, sem espaços e sem começar por número. A linguagem C diferencia letras maiúsculas de minúsculas.

  • idade e Idade são nomes diferentes.
  • nota_final é melhor que nf para iniciantes.
  • 2valor é inválido porque começa com número.

Exercícios rápidos

  1. Crie variáveis para nome, idade e altura.
  2. Calcule a média de três notas usando float.
  3. Crie uma constante para o valor de uma passagem de ônibus.
Dica: escolha o tipo de dado apropriado para cada situação. Isso melhora o uso de memória e evita erros.
