Tipos básicos de dados

As variáveis armazenam valores. Os tipos mais comuns são:

  • int - números inteiros
  • float - números de ponto flutuante
  • double - números de ponto flutuante mais precisos
  • char - um único caractere
int idade = 20;
float altura = 1.75f;
double peso = 68.5;
char letra = 'A';
Dica: Escolha o tipo de dado apropriado para cada situação. Isso garante melhor uso de memória e evita erros.
