Estrutura de um Programa em C

Entenda como os programas em C são organizados.

Início Estrutura Variáveis Entrada/Saída Condicionais Laços

Um programa C básico usa a função main() como ponto de entrada

O código abaixo exibe uma mensagem no console:

#include <stdio.h>

int main(void) {
    printf("Olá, C!\n");
    return 0;
}

Explicação:

  • #include <stdio.h> traz a biblioteca padrão de entrada e saída.
  • int main(void) declara a função principal.
  • printf imprime texto na tela.
  • return 0; indica que o programa terminou com sucesso.
Dica: Todo programa em C deve ter uma função main() que é executada quando o programa inicia.
