Um programa C básico usa a função main()

A função main é o ponto de entrada do programa. Quando o executável inicia, é por ela que a execução começa.

#include <stdio.h>

int main(void) {
    printf("Ola, C!\n");
    return 0;
}
  • #include <stdio.h> traz a biblioteca padrão de entrada e saída.
  • int main(void) declara a função principal.
  • { } delimitam o bloco de comandos da função.
  • printf imprime texto na tela.
  • return 0; indica que o programa terminou com sucesso.

Partes importantes do código

Em C, cada detalhe da sintaxe importa. Ponto e vírgula, chaves e parênteses ajudam o compilador a entender o que deve ser executado.

Elemento Função
#include Inclui bibliotecas com funções prontas.
main Função principal do programa.
; Finaliza uma instrução.
// Cria um comentário de uma linha.
/* */ Cria um comentário de várias linhas.

Comentários e organização

Comentários servem para explicar decisões do código. Eles são ignorados pelo compilador, mas ajudam outras pessoas a entenderem o programa.

#include <stdio.h>

int main(void) {
    // Mostra uma mensagem inicial
    printf("Bem-vindo ao curso de C!\n");

    /*
       Este comentário pode ocupar
       mais de uma linha.
    */
    return 0;
}

Como compilar

Se você estiver usando o compilador GCC, salve o arquivo como programa.c e compile pelo terminal:

gcc programa.c -o programa
./programa

No Windows, o executável pode ser iniciado com:

programa.exe

Exercícios rápidos

  1. Crie um programa que mostre seu nome na tela.
  2. Adicione dois printf: um para o nome e outro para a cidade.
  3. Coloque um comentário explicando o objetivo do programa.
Dica: todo programa em C deve ter uma função main() que é executada quando o programa inicia.
