Condicionais em C

Tome decisões em seus programas com estruturas condicionais.

Início Estrutura Variáveis Entrada/Saída Condicionais Laços

Decisões com if

O if executa um bloco apenas quando a condição é verdadeira.

int idade = 18;

if (idade >= 18) {
    printf("Maior de idade\n");
}

if, else if e else

Use essa estrutura quando há mais de uma possibilidade de resposta.

int a = 10;

if (a > 0) {
    printf("a e positivo\n");
} else if (a < 0) {
    printf("a e negativo\n");
} else {
    printf("a e zero\n");
}

Operadores de comparação

Operador Significado Exemplo
== Igual a x == 10
!= Diferente de x != 0
> Maior que x > 5
< Menor que x < 5
>= Maior ou igual x >= 18
<= Menor ou igual x <= 100

Operadores lógicos

Operadores lógicos combinam condições para criar regras mais completas.

int idade = 20;
int tem_documento = 1;

if (idade >= 18 && tem_documento == 1) {
    printf("Entrada permitida\n");
}
  • && significa "e": as duas condições precisam ser verdadeiras.
  • || significa "ou": pelo menos uma condição precisa ser verdadeira.
  • ! significa "não": inverte o resultado da condição.

Escolhas com switch

O switch é útil quando você compara uma variável com vários valores específicos.

int opcao = 2;

switch (opcao) {
    case 1:
        printf("Cadastrar\n");
        break;
    case 2:
        printf("Listar\n");
        break;
    case 3:
        printf("Sair\n");
        break;
    default:
        printf("Opcao invalida\n");
}

O break impede que a execução continue para os próximos casos.

Exemplo prático: aprovação

float media = 6.8f;

if (media >= 7.0f) {
    printf("Aprovado\n");
} else if (media >= 5.0f) {
    printf("Recuperacao\n");
} else {
    printf("Reprovado\n");
}

Exercícios rápidos

  1. Leia uma idade e informe se a pessoa pode votar.
  2. Leia um número e diga se ele é par ou ímpar.
  3. Crie um menu com switch para escolher entre somar, subtrair e sair.
Dica: use == para comparar igualdade. O operador = é usado para atribuir valor.
