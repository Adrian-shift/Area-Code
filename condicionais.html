Condicionais em C

Tome decisões em seus programas com estruturas condicionais.

Início Estrutura Variáveis Entrada/Saída Condicionais Laços

O if permite tomar decisões

Use a estrutura if...else if...else para controlar o fluxo do programa:

int a = 10;
if (a > 0) {
    printf("a é positivo\n");
} else if (a < 0) {
    printf("a é negativo\n");
} else {
    printf("a é zero\n");
}
Dica: Use operadores de comparação (<, >, ==, !=) e operadores lógicos (&&, ||, !) para criar condições complexas.
