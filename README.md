# Programação em C - Tutorial Interativo

![Visual do site](https://github.com/user-attachments/assets/6b86a846-744c-4b9d-adf6-5e8a55180079)

Um site educativo em HTML e CSS que apresenta um tutorial introdutório de programação em C. O projeto organiza os principais tópicos da linguagem em páginas separadas e oferece um design responsivo para o usuário navegar em desktop e dispositivos móveis.

## Estrutura do Projeto

- `index.html` — página inicial com apresentação do conteúdo e navegação principal.
- `estrutura.html` — explica a estrutura básica de um programa em C.
- `variaveis.html` — aborda variáveis, tipos de dados e declarações.
- `entradasaida.html` — trata de entrada e saída de dados em C.
- `condicionais.html` — cobre estruturas condicionais como `if` e `switch`.
- `lop.html` — mostra laços de repetição como `for`, `while` e `do-while`.
- `estilo.css` — define o estilo visual e as regras de responsividade.

## Recursos principais

- Layout limpo e moderno com cabeçalho colorido para cada tópico
- Navegação fixa e intuitiva para mudar entre as páginas
- Design responsivo para telas pequenas, incluindo celulares
- Conteúdo organizado para estudo rápido dos conceitos

## Como usar

1. Acesse o [aqui](https://adrian-shift.github.io/Area-Code/) no navegador.
2. Clique nos links de navegação para acessar os tópicos.
3. Use a visualização no celular para testar o layout responsivo.

## Personalização

- Edite `estilo.css` para ajustar cores, tipografia e espaçamentos.
- Adicione imagens ou exemplos de código nas páginas `.html` para enriquecer o conteúdo.
- Atualize o texto das páginas com mais explicações conforme o material evolui.

## Sobre

Este projeto é ideal para estudantes iniciantes que desejam um guia visual e organizado para os primeiros passos em programação em C. O site pode ser usado localmente ou como base para um portfólio de estudos.
